To display simple alert message in selected posts.

== Description ==

This plugin display simple alert message in posts, pages as well as custom post type.

Plugins allow you to select specific post, page or custom post type from plugin admin settings and display your custom message on front.

A few notes:

*   Support inbuild post type such as posts and pages.
*   Support custom post type such as products (woo) etc.
*   "Requires at least 4.5 wp version" is the lowest version that the plugin will work on
*   "Tested up to 5.3.2" is the highest version that you've *successfully used to test the plugin*. Note that it might work on
higher versions... this is just the highest one you've verified.

== Installation ==

To install the plugin and get it working follow below steps.

1. Upload `simple-alert` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin from Settings->Simple Alert
